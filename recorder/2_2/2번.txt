동적할당
- new
  메모리 할당
  클래스도 동적할당이 가능
  자료형의 크기만큼 포인터 변수에 메모리 할당
  + int* a = new int (1);
     초기값 지정 가능
  + int* arr = new int [5] { 1, 2, 3, 4, 5};
     배열도 초기값 지정 가능
- delete
  메모리 해제
  배열을 할당한 경우, delete[]를 사용

4번
(a)
- Dynamic Programming (동적 계획법)
  하나의 큰 문제를 여러개의 작은 문제로 나누어서 그 결과를 저장하여 다시 큰 문제를 해결할 때 사용
  (a) 비재귀 사용
      반복문을 사용하여 반복적으로 부분 문제들을 해결, 결과를 배열에 저장
  (b) 재귀 사용
       기저 상태에서 출발하지 않고, 위에서부터 바로 호출을 시작
       결과 값을 재귀를 통해 전이시키는 방식
